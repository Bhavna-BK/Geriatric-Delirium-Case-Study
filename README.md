# Geriatric Delirium: Cognitive and Neurological Impairments Linked to Polypharmacy

**Author:** Bhavna Kakkar  
**Publication Year:** 2024

---

## 1. Project Purpose and Overview

This repository serves as the official, accessible source for the published academic case study titled **"Geriatric Delirium: Cognitive and Neurological Impairments Linked to Polypharmacy."**

The primary purpose of hosting this repository is to enhance the **discoverability** of the paper, making it easily searchable and accessible via platforms like GitHub and general search engines, as it is not readily visible through common search methods.

### Abstract Summary

This report examines the complex case of an 80-year-old female patient who developed **delirium** secondary to **polypharmacy**, underlying medical conditions (e.g., **hyponatremia**), and the stress of hospitalization. The study provides a detailed account of the acute symptom onset, progression (including **hyperactivity, hypoactivity, hallucinations,** and **memory/cognitive distortion**), and the critical treatment and resolution administered from admission.
